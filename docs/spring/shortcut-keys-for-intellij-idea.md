---
title: Shortcut keys for IntelliJ IDEA
date: 2018-10-04T07:14:45.721Z
---
* cmd + p (Show tooltip for arguments etc)
* cmd + options + L (Format code)
* ctrl + option + O (Optimize imports)
* cmd + shift + U (Toggle Case)
* ctrl + G (Multiselect)
